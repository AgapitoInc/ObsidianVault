# Obsidian Project Management Vault Template

A pre-configured Obsidian vault for engineering project management with clients, projects, tasks, proposals, and daily notes.

## Getting Started

1. Open this folder as a vault in Obsidian
2. All plugins are pre-installed — accept any prompts to enable community plugins
3. Add team members to `People.md`
4. Use the buttons on `_MAIN.md` (the dashboard) to create new clients, projects, tasks, and proposals

## Structure

| Folder | Purpose |
|--------|---------|
| `Clients/` | One note per client |
| `Projects/` | Project notes, organized by client subfolder |
| `Tasks/` | Individual task notes |
| `Proposals/` | Proposal notes |
| `Office/` | Internal/office notes |
| `Archive/` | Archived items |
| `DailyNote/` | Auto-generated daily notes |
| `Templates/` | Templater templates (do not edit unless customizing) |

## Key Files

- **`_MAIN.md`** — Dashboard with dataview tables for tasks, projects, budgets
- **`People.md`** — Team roster; names here are used in task `assignee` fields

## Plugins Included

- Dataview — dynamic tables and queries
- Templater — template engine for new notes
- Meta Bind — buttons on the dashboard
- Periodic Notes / Natural Language Dates — daily note support
